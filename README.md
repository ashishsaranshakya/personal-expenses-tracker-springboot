# Personal Expense Tracker – Spring Boot Backend

This repository contains the **Spring Boot backend implementation** of the Personal Expense Tracker application.

It is a complete rewrite of the original Express.js backend using **Java and Spring Boot**, while keeping the **same API contract and endpoints**.

🔗 Original MERN Full Stack Repository:
[ashishsaranshakya/personal-expenses-tracker](https://github.com/ashishsaranshakya/personal-expenses-tracker)

---

## Overview

This service provides REST APIs for managing:

* User authentication (Google OAuth)
* Expenses
* Incomes
* Categories
* User profile & balance

The API structure matches the original Node.js backend, making it fully compatible with the existing frontend.

API Documentation: <a href="/API-Doc.md">API Documentation</a>

---

## Tech Stack

* Java 17+
* Spring Boot
* Spring Web (REST)
* Spring Security
* OAuth2 (Google Login)
* Spring Data JPA
* MongoDB
* Maven

## Features

* Google OAuth2 authentication
* Secure REST APIs
* CRUD operations for expenses and incomes
* Custom category management
* User profile & balance management
* Database persistence with JPA
